TIME ATTACK 모드

4개 세부 모드 — NUT HAND NORMAL · HARD · EXTRA HARD · SIDE POT — 각각 별도의 시작 시간과 보너스 밸런스로 진행. 모드별로 로컬 top-10 리더보드 가 영구 저장되어 점수와 스테이지, 날짜를 확인할 수 있습니다.

  • 시작: NORMAL·SIDE POT 60s · HARD 90s · EXTRA HARD 120s
  • 정답 +5초 (EXTRA HARD 는 +7초), 오답 −3초
  • HARD / EXTRA HARD 는 1st/2nd/3rd 줄당 채점 — 2/3 정답이면 +5+5−3 = +7s 순이익
  • SIDE POT 은 5점 도달 시 자동으로 HARD 시나리오 (3+명 + 총 팟 숨김) 으로 전환
4모드 chooser — 각 모드에 BEST 점수 표시 + LEADERBOARD 진입

SPEED UP! 시스템

스테이지 10마다 시계가 빨라집니다. 0-9 스테이지는 정상 속도, 이후 ×1.2 → ×1.4 → … → ×2.0 까지 가속하며, 스테이지 50 이후 cap 적용으로 더 빨라지지는 않습니다. 전환 시 화면 전체에 반투명 빨간 wash 가 깔리며 중앙에 "SPEED UP!" 이 표시됩니다.

×1.2 진입 순간 — 정답 결과 위로 빨간 wash 가 깔리며 중앙에 SPEED UP! 표시

3-2-1-GO! 카운트다운

TIME ATTACK 진입 시 게임 시작 전 카운트다운이 표시됩니다. 보드를 미리 확인할 시간을 확보하고, 갑작스러운 시작으로 인한 당황을 줄였습니다.

NUT HAND HARD 진입 직후 — 보드는 보이지만 입력은 차단 (AbsorbPointer)

게임 화면 (HARD / EXTRA HARD)

카운트다운 직후 진입하는 인-게임 화면. 상단에 TIME ATTACK HUD (남은 시간, 스테이지, 점수), 그 아래 community 5장 + TOP 3 NUT HANDS 슬롯 3행 + 덱 그리드 + SUBMIT. 채점 직후 시계는 자동 일시정지.

90초 시작 · STAGE 1 · 줄당 채점 모드

로컬 리더보드

모드별 탭으로 전환되는 top-10 점수판. 점수 / 스테이지 / 날짜가 저장되며, 모드별로 개별 초기화도 가능합니다. 전체 인터넷 권한 없이 기기 안에서만 보관됩니다.

모드별 탭 + 빈 상태 메시지 ("No records yet")

시작 화면 — PLAY 섹션 카드 2 → 3

기존 NUT HAND + SIDE POT 두 BigCard 아래에 TIME ATTACK 카드가 추가되었습니다. 각 카드 높이가 약간 줄었지만 가독성은 유지. TIME ATTACK 카드는 타이머 아이콘과 딥오렌지 accent 로 다른 두 카드와 시각적으로 구분됩니다.

PLAY: NUT HAND / SIDE POT / TIME ATTACK · LEARN: TUTORIAL / HAND RANKINGS

NORMAL TIME ATTACK 페어드 보드 확률 ↓

NORMAL 모드에서 커뮤니티 5장 중 같은 랭크 2장 이상이 나오는 경우의 확률을 약 49% → 약 15% 로 낮췄습니다. 페어 보드는 "포켓 페어 = 너트" 같이 반복되기 쉬워서, 빠른 페이스에서 너트 퀴즈가 더 다양해지도록 조정. 완전 제거는 아닙니다 — 자연스러운 분포는 유지됩니다.

채점 결과 표시 시간 연장

정답 2.0초 / 오답 3.0초 (이전 0.7~1.8초). TIME ATTACK 한정으로 결과 표시 동안 시계가 정지 하므로, 점수 손해 없이 충분히 결과를 읽을 수 있습니다. HARD/EXTRA 의 3행 결과 + 예시카드도 여유 있게 확인.

NEW 버튼 (TIME ATTACK 한정) 숨김

NUT HAND / SIDE POT 둘 다 TIME ATTACK 모드에서는 NEW (재배포) 버튼이 사라집니다. 어려운 보드를 무료로 건너뛰는 게 점수제와 어울리지 않기 때문에, submit 후 자동 advance 만이 유일한 진행 경로입니다.

FIXED · 수정

카운트다운 중 입력 차단

TIME ATTACK 진입 후 카운트다운이 끝날 때까지 AbsorbPointer 로 입력이 차단됩니다. 실수로 미리 카드를 탭하거나 SUBMIT 을 눌러 다음 스테이지로 넘어가는 케이스가 사라졌습니다.

결과 표시 동안 timer pause

이전에는 채점이 표시되는 동안에도 시계가 계속 흘렀습니다. 이제 submit 직후 controller 를 일시정지하고, 다음 보드가 deal 될 때 재개됩니다. 결과를 읽는 시간이 점수에 영향을 주지 않습니다.

HAND RANKINGS 페이지

10개 포커 족보를 강한 순서대로 나열하고, 각 항목에 예시 카드 5장 + 한국어/영어 설명을 제공. 별도 섹션으로 사이드 팟 분배 규칙 워크스루도 포함되어, 메인 팟 / 사이드 팟 작동 원리를 실제 예시로 학습할 수 있습니다.

사이드 팟 규칙 + 예시 + #1 Royal Flush 부터 강한 순서대로

인터랙티브 튜토리얼 (스팟라이트 오버레이)

각 모드 첫 진입 시 게임 화면 위에 스팟라이트로 가이드가 자동 안내됩니다. 메인 메뉴의 TUTORIAL 카드에서 언제든 재시청 가능 (NUT HAND / SIDE POT / EXTRA HARD 선택).

  • NUT HAND — 커뮤니티 카드 / 슬롯 / 덱 / 액션 / 상단 영역 5스텝
  • SIDE POT — 총 팟 배너 / 플레이어 행 / 넘패드 / 제출 4스텝
Step 1/5 — 커뮤니티 카드 영역에 스팟라이트 + 캡션 + NEXT/SKIP

EXTRA HARD 튜토리얼

게임 중 첫 활성화 시 자동 발화 (4스텝): 제약 칩 의미, 칩 탭으로 상세 보기, TOP 3 nut 재계산, 위반 시 자동 오답. 튜토리얼 탭에서 선택하면 EXTRA HARD 상태로 강제 전환 → 안내 → 끝나면 이전 상태로 복원되고 자동으로 홈으로 복귀.

오답 시 정답 핸드 예시 카드 (NUT HAND)

Easy 에서는 BEST POSSIBLE 아래에 정답 핸드를 만드는 홀카드 2장 표시. Hard 에서는 틀린 줄마다 정답 핸드의 작은 예시 카드 표시. "Two Pair" 같은 추상 텍스트 대신 실제 카드 조합을 즉시 시각 확인할 수 있습니다.

FIXED · 수정

작은 폰에서 1.1px 오버플로

Container.Border.all() 의 1dp 폭이 자식 영역을 추가로 잠식하는 것을 metric 계산에 반영. 320×568급 폰에서 NUT HAND 커뮤니티 / 덱 영역이 오버플로 나던 문제 해소. 해당 컨테이너에 ClipRRect 방어막을 추가해 미래의 sub-pixel 이슈도 방지.

언어 토글이 일부 화면에서 즉시 반영 안 되던 문제

NUT HAND 미니 게임

커뮤니티 5장이 주어졌을 때 가능한 최강 핸드 (Nut) 를 만드는 홀카드 2장을 고르는 퀴즈. Normal · Hard · Extra Hard 세 단계 난이도.
